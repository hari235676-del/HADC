Module 4 — Arithmetic coding entropy layer
Arithmetic coding post-processing: the hybrid token stream still contains residual statistical redundancy. An arithmetic coder assigns fractional-bit codes to tokens based on their empirical probability, approaching the Shannon entropy lower bound. This is applied as a final pass — it does not alter the dictionary or token logic above.
